# ASCII Matrix

# Installation
## Linux (Ubuntu 22.04)
```
sudo apt update
sudo apt install libopencv-dev
```

# Compilation
## Linux (Ubuntu 22.04)
In _source/_ directory:
```
cd build && cmake .. && make && cd ..
```

# Usage (console test)
## Linux (Ubuntu 22.04)
In _source/_ directory:
```
# capture webcam
./build/asciiMatrix

# capture video file
./build/asciiMatrix data/sample.mp4
```
